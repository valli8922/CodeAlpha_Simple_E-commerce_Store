# Simple E-commerce Store

A full-stack e-commerce website developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

- User Registration & Login
- Product Listing
- Product Details Page
- Shopping Cart
- Quantity Management
- Order Processing
- Search Functionality
- Category Filtering
- Dark Mode
- Responsive Design

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB
